# Google Play Store Assets

This directory contains all promotional graphics and assets for the Google Play Store listing.

## Files Included

### Required Assets

1. **app-icon-512x512.png** (512x512 pixels)
   - High-resolution app icon for Play Store
   - Used in search results and app listing

2. **feature-graphic-1024x500.png** (1024x500 pixels)
   - Banner graphic displayed at top of Play Store listing
   - Shows app name, tagline, and preview

3. **screenshot-1-config.png** (1200x1920 pixels)
   - Configuration screen showing pairing token interface
   - Demonstrates easy setup process

4. **screenshot-2-display.png** (1200x1920 pixels)
   - Main event display with "NOW" card highlighted
   - Shows real-time schedule with upcoming events

5. **screenshot-3-events.png** (1200x1920 pixels)
   - Full event list view with past and upcoming events
   - Demonstrates facilitator photos and event details

## Upload Instructions

1. Go to Google Play Console
2. Navigate to **Store presence > Main store listing**
3. Scroll to **Graphics** section
4. Upload each asset to its corresponding field:
   - App icon → "App icon" (512x512)
   - Feature graphic → "Feature graphic" (1024x500)
   - Screenshots → "Phone screenshots" or "Tablet screenshots" (1200x1920)

## Notes

- All images are optimized for Google Play requirements
- Screenshots show realistic app usage scenarios
- Graphics use consistent branding and color scheme
- All assets are PNG format for best quality
